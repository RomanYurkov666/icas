IntroductionÂ¶

This document describes the iCAS Sellside API to place ads on iCAS.

The API is designed as a RESTful API supporting JSON as data formats. Authentication is provided va OAuth2 to allow simple integration with existing client libraries.

If you have questions or would like to provide feedback you can post a message on eBay Classifieds iCAS Community.
